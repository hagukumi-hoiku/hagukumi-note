職員マスタ
📋 職員マスタ 入力ガイド
・ユーザー名：半角英数字のみ（例：kasugai）
・パスワード：半角英数字のみ（例：pass1234）
・氏名：苗字＋（名前の頭文字ひらがな1文字）
　例）春日井太郎 → 春日井（た）、同姓がいる場合は区別できるよう入力
・権限：管理者 / 園長 / 主任 / 統括リーダー / 職員 から選択
・有効：在職中は ○、退職者は ×（データは残ります）
・担任クラスの設定は「クラス・担任マスタ」で行います
氏名（ユーザー名） 役割 有効 操作

クラス・担任マスタ
ℹ️ クラス名はスプレッドシートの マスタ_クラス と連動しています。変更後は自動保存されます。
年齢：0歳=ひよこ相当、1歳=りす相当…乳児(0〜2歳)と幼児(3〜5歳)でタスク種別が変わります。
クラス名 年齢 担任 在籍数 操作

様式ごとの承認者設定
ℹ️ 各様式の第1承認者（主任 または 統括リーダー）を設定します。最終承認は常に 園長 です。

年間指導計画 期の設定
ℹ️ 年間指導計画の各期（第1期〜第4期）の月範囲を設定します。年案入力画面のタイトルに反映されます。
期 開始月 終了月 表示例

第2期 8月〜11月
第3期 12月〜2月
